# MoMo

Projeto estático de demonstração de um fluxo simples de login e reserva de aula.

## Descrição

Este projeto é um site estático em HTML, CSS e JavaScript que simula:

- Tela de login (`index.html`)
- Listagem de aulas disponíveis (`home.html`)
- Confirmação de reserva (`reserva.html`)

O objetivo é demonstrar uma navegação básica entre páginas e estilos responsivos.

## Estrutura de arquivos

- `index.html` - página inicial de login
- `home.html` - página com aulas disponíveis para reserva
- `reserva.html` - página de confirmação de reserva
- `css/style.css` - estilos do site
- `js/script.js` - scripts para navegação entre páginas

## Como usar

1. Abra o arquivo `index.html` em um navegador.
2. Na tela de login, clique em `Entrar` para ir para a página de aulas.
3. Escolha uma aula e clique em `Reservar` para ver a confirmação.

## Tecnologias

- HTML
- CSS
- JavaScript

## Observações

- O login não valida credenciais reais.
- A navegação entre páginas é feita apenas por redirecionamento com JavaScript.
- O layout é responsivo para telas menores.
